## bfs
通常用队列，先进先出FIFO实现
1. 初始化队列Q
2. Q={起点s};标记起点s为已访问
3. while (Q非空) {
    取Q队首元素u； u出队；
    if (u === 目标) {.. sth}
    所有与u相邻且未被访问的点进入队列
    标记u为已访问
}

## dfs
一直住深处走，直到走不下去或找到解
DFS：使用栈保存未被检测的结点，结点按照深度优先的次序被访问并依次被压入栈中，并以相反的次序出栈
dfs(depth) {
    if (找到解|走不下去) {
        return
    }

}
